# Loudess - Bunny Volume Control Plugin

## Description
A fast and lightweight Bunny plugin for Discord that provides:
- **Volume Control** - Adjust Discord audio volume with `/vol` command
- **Fake Deafen** - Toggle fake deafen status with `/fakedeafen` command

## Installation
1. Clone this repository
2. Place it in your Bunny plugins directory
3. Restart Bunny

## Usage

### Volume Control
```bash
/vol 50
```
Sets volume to 50%. Range: 0-100

### Fake Deafen
```bash
/fakedeafen
```
Toggles fake deafen on/off
